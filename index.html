CommuniTask

January—May, 2015

Junior project, worked with four others.
CommuniTask is a group task management system with a gamification element. Each group sets a time period in which the tasks must get done, and for each task completed each user may progress their piece on the board a weighted dice roll number of spaces.

CommuniTask's prototype was made as a mobile-optimized website, mainly using HTML. CSS, JS, and jQuery. It was intended to be a UX prototype, rather than a fully functional site.

Contributions: collaborator in developing the concept of our project, as well as working through the nuances of how it should be set up and run. Lead programmer on the prototype.

Linguistic Self-Expression on Dating Websites

August 2015—May 2016

Senior project, working with K Sadwin.
This project is an analysis of use of language by gender on dating websites (specifically OKCupid). We used Python 3 with the libraries Beautiful Soup and textBlob to scrape and parse over 1500 user profiles and stored the content in a SQLite database.

We are looking for patterns in language use by demographic groups, with a current focus on gender and sexuality. We have generated word clouds and tf-idf word clouds to view distinctions and variations in the content. The tf-idf clouds are on Sadwin's site.

Award finalist at the 2016 James J. Whalen Academic Symposium, and presented at the 2016 New York Celebration of Women in Computing

Byte Bots

November—December, 2015

Working with a partner, final project for Multimedia Programming

A game based on Rock 'Em Sock 'Em Robots, created using HTML, CSS, JavaScript, and specifically the <canvas> element.

We chose to use objects for the robots instead of local variables because we wanted to set each function specific to which way the robots were facing, and defining them within the robot object simplified that process greatly.

Contributions: the interactions on the page as well as the attack and update health functions.

DnD Character

November—December, 2015

Four-person term project for Data Structures

We wrote a program that assists users in making a character object, and then saves that data to a file for the user to reference later. This program was written in C++ and collaborated on via GitHub.

Contributions: the inventory that a user can create. The inventory was backed by a linked list, and could be saved and loaded to and from a separate file. Users could add, remove, and sell items in their inventory.

HVZ Web App

January—May 2016

A project using the Java Play framework for web development.
Made for a Software Engineering course, the finished product allows players to send messages to all other players who are also humans or zombies. Players are able to switch from human to zombie when bitten, and Moderators are able to track the number of humans and zombies in their game and send messages to all players.

Contributions: documentation, diagrams, and database structure. Initialization of and connection to a MySQL database. Connections between back- and front-end components and functions.

Mind Body Soul

September—December 2014

A mental health tracking application created for Advanced Web Programming.

Working in a group of three, we created a web application which allowed users to input activities and moods to track trends and correlations. The site also included mental health resources and a section designed to help users in crisis.

Contributions: user interface, database architecture, some server-side functionality.
